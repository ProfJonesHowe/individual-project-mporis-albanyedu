A CYBR201 Poem

At last the end has come

For CYBR201

But let's reflect on some

Of the things that we've done

We learned about the web

Design and history

We learned HTML

The basics made easy

Format with CSS

Websites pleasant to see

It helps avoid a mess

Mobile friendliness key

We did some JavaScript

Making functions is fun

Now we are well equipped

To design at least one

We did a group project

Presented at Showcase

It had a good effect

Inside the Cyber Space

We also did some labs

To learn how to make sites

But now we close the tabs

And end these few highlights
